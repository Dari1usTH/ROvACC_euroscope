Installation Steps:

    1)Create a directory and move DiscordEuroscope.dll file into it.
    (Recommended path: LRBB\Plugins\YourFolderName)

    2)Open EuroScope, go to OTHER SET > Plug-ins, and load the .dll file.

Enjoy !
